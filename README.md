# Aplikasi Penyewaan Tempat

Aplikasi web sederhana (tanpa backend) untuk mengelola data tempat sewa secara dinamis dan mencatat transaksi penyewaan.

## Fitur

- Tambah tempat baru (nama, lokasi, harga per hari, deskripsi).
- Daftar tempat tersimpan dan bisa dihapus.
- Form penyewaan dengan perhitungan total otomatis.
- Riwayat penyewaan tersimpan.
- Data disimpan di `localStorage` browser.

## Menjalankan

Cukup buka file `index.html` langsung di browser.

Atau gunakan server lokal sederhana:

```bash
python3 -m http.server 8000
```

Lalu akses `http://localhost:8000`.
